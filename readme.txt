== Description ==

SD live seacrh is a basic search plugin for search lively.

Basic Usage

Activate the SD live Search plugin
Add the shortcode [wpbsearch] to a page or something


--
Example:

`[wpbsearch]`


== Installation ==
1. Navigate to 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `sd-live-search.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
